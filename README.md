# CoCo Connect HDL Releases

Release artifacts for `coco-connect-hdl`.

This repository stores versioned Windows binaries and the matching HDL SKILL bridge file. Keep `coco-connect-hdl.exe` and `bridge.il` in the same release version because their IPC protocol must match.

## Layout

```text
vX.Y.Z/
  windows-x64/
    coco-connect-hdl.exe
  skill/
    bridge.il
latest.json
```
